PETUNJUK: SIMPAN LOGO DISINI
================================

Simpan file logo KarWanua dengan nama:
  karwanua-logo.png

Di folder ini (public/).

Spesifikasi Logo:
- Format: PNG (transparant background)
- Ukuran: 200x200 px atau lebih besar
- Rasio: 1:1 (square)
- File size: < 100KB

Setelah menyimpan logo, restart server:
  npm run dev
